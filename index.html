連接 Supabase 雲端同步

串接您的 Supabase 資料庫後，您就可以在電腦與 iPad 之間即時同步所有的手繪與文字！

💡 如何取得？

  1. 登入 Supabase 官網 並建立一個新專案。
  2. 在專案的 Project Settings > API 頁面找到 Project URL 與 anon public Key。
  3. 在專案的 SQL Editor 貼入以下 SQL 碼來建立資料表（點擊下方按鈕可複製 SQL）：
CREATE TABLE IF NOT EXISTS notebook_pages (
  id TEXT PRIMARY KEY,
  page_num INTEGER NOT NULL,
  drawings JSONB DEFAULT '[]'::jsonb,
  texts JSONB DEFAULT '[]'::jsonb,
  is_shared BOOLEAN DEFAULT false,
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now())
);

-- 啟用即時同步 (Realtime)
alter publication supabase_realtime add table notebook_pages;

📚 使用與手寫說明

✍️ 雙層編輯系統

  • 手繪層（頂部）：可直接使用滑鼠、電繪板，或在 iPad 上使用 Apple Pencil 與手指直接於紙張上進行流暢繪圖或書寫。
  • 文字層（底部）：在紙張的任意空白處點擊，即可在該位置新增文字輸入框。輸入完成後，可任意拖曳其位置，或者雙擊該文字框將其刪除。

📖 擬真翻頁

  • 您可以使用左右導覽按鈕翻頁，或直接在書本邊角點擊並拖曳，享受擬真 3D 翻書效果。
  • 首頁為筆記本封面，末頁為封底。中間頁面為可編輯與分享的內頁。

🔗 頁面公開分享

  • 每頁皆可獨立開啟分享開關。開啟後會自動生成唯讀公開連結，複製後分享給他人，他人將可透過此連結即時檢視您的筆記與手寫（無法編輯）。關閉開關後分享連結即刻失效。
